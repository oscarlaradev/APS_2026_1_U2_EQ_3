# APS_2026_1_U2_EQ_3

Proyecto de ejercicios de C# (.NET) para la Unidad 2.

## Estructura

- Carpeta principal: [Clase_ICDIA_Unidad2](Clase_ICDIA_Unidad2)
- Punto de entrada: [Clase_ICDIA_Unidad2/Program.cs](Clase_ICDIA_Unidad2/Program.cs)
- Proyecto: [Clase_ICDIA_Unidad2/Clase_ICDIA_Unidad2.csproj](Clase_ICDIA_Unidad2/Clase_ICDIA_Unidad2.csproj)
- Handlers (lógica): [Clase_ICDIA_Unidad2/HandlersEjerciciosBasicos](Clase_ICDIA_Unidad2/HandlersEjerciciosBasicos)
- Runners (ejecución): [Clase_ICDIA_Unidad2/EjecutoresDeEjercicios](Clase_ICDIA_Unidad2/EjecutoresDeEjercicios)
- Auxiliares: [Clase_ICDIA_Unidad2/auxiliar](Clase_ICDIA_Unidad2/auxiliar)

## Ejercicios disponibles

1. Par o impar
2. Suma A + B
3. Invertir X y Y
4. Comprobar mayoría de edad
5. A y B: número mayor
6. Multiplicación A * B
7. Suma A + B genérica
8. Cantidad de efectivo
9. Índice de masa corporal (IMC)
10. Distancia entre dos puntos
11. Promedio de 3 calificaciones
12. Número primo
13. Factorial

## Cómo ejecutar

Desde la carpeta [Clase_ICDIA_Unidad2](Clase_ICDIA_Unidad2):

```bash
dotnet run
```

Después, selecciona el número del ejercicio en el menú.
